0、前言

公司计划下一个版本的PC和wap网站使用node.js技术，正在学习node＋MongoDB的使用。

P.S 本文安装环境是OS X 10.10.4 Yosemite 优胜美地。

1、安装HomeBrew

什么是HomeBrew？Homebrew官网

Homebrew installs the stuff you need that Apple didn’t.

-----Homebrew 使 OS X 更完整。

终端输入


	$ ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

安装成功显示


==> Installation successful!
==> Next steps
Run `brew help` to get started
wuguanxideMacBook-Pro:~ wuguanxi$ brew -v
Homebrew 0.9.5

2、安装MongoDB

更新Homebrew的package数据库，在Mac的终端中输入：


	$ brew update

更新成功后，在Mac的终端中输入：


	$ brew install mongodb

耐心等待然后会看到安装成功：


	==> Downloading https://homebrew.bintray.com/bottles/mongodb-3.0.4.yosemite.bottle.tar.gz
	######################################################################## 100.0%
	==> Pouring mongodb-3.0.4.yosemite.bottle.tar.gz
	==> Caveats

	To have launchd start mongodb at login:
	    ln -sfv /usr/local/opt/mongodb/*.plist ~/Library/LaunchAgents
	Then to load mongodb now:
	    launchctl load ~/Library/LaunchAgents/homebrew.mxcl.mongodb.plist
	Or, if you don't want/need launchctl, you can just run:
	    mongod --config /usr/local/etc/mongod.conf
	==> Summary
	🍺  /usr/local/Cellar/mongodb/3.0.4: 17 files, 154M

3、启动MongoDB

启动前要先在 “/” 目录下新建/data/db 文件夹


	$mkdir -p /data/db

然后进入mongodb的bin文件夹执行命令“./mongod”启动mongoDB server


	$ cd /usr/local/Cellar/mongodb/3.0.4/bin
	$ sudo ./mongod

4、MongoDB的控制台

另外打开一个终端窗口，同样为了方便起见，进入"bin"目录，运行命令"./mongo"


	$./mongo
